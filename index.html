 Java 组介绍：

人的一生不能两次踏进同一条河流，但是服务器不能在连续两次上线的时候都崩掉！我们保证，绝不让服务器挂掉！

数据更新与功能实现是我们的职责，不修边幅、低调潜行是我们的作风。

我们有持续的学习动力，喜欢享受解决问题的快感。最重要的是，我们看到了自己每一天在技术上的进步。

我们没有前端效果的狂拽酷炫屌，但是我们一直坚持，默默在幕后努力。要知道，能被一棍子打死的都没有后台。

不用犹豫 ，赶快加入我们的阵营吧。

 C++ 组介绍：

当你发现你想换电脑的时候，就来我们组吧。

有一天你发现你笔记本的摄像头坏了，你发现 8G 的内存一下就完了，你发现 1T 的硬盘程序没跑几分钟就塞满了。


当你想把自己的脑子给换掉，就来我们组吧。

一大堆的宏要去记住，视频格式要去了解，接口要调用来调用去。不是你崩溃就是你电脑崩溃。


既然如此，那我们为啥还选择 C++ ？因为我们热爱挑战！
